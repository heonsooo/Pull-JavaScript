## 정의
- 프로그래밍 언어에서 함수라고 하는 코드의 묶음을 일반적인 값처럼 취급하는 개념이다.
  - 함수를 변수에 넣는 것.

## 특징
- 일반적인 값처럼 취급한다 = 일반적인 값처럼 **변수**에 넣을 수 있다.
- 변수에 넣는다. 라는 의미는 다양한 쓰임이 있을 수도 있지 않을까 라는 예상을 할 수 있다.


일반적인 변수 : 함수 안쪽에 지역 변수로서 변수가 등장하지만, 인자로서도 변수가 등장한다.
리턴하는 값도 함수가 될 수 있다.
이처럼 변수는 굉장히 다양한 쓰임이 있다.

이때 이 변수를 함수로 할당할 수 있게 된다면, 
- 인자로 받는게 값이 데이터가 아니고 함수. 
- 리턴 되는 값이 데이터가 아니고 함수.


일급함수는 다양한 쓰임, 용도, 테크닉이 존재한다.
